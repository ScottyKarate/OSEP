# Enumeration

### Enumerate AD for SPN's with built in setspn

``` Powershell
setspn -T corp1 -Q MSSQLSvc/*
```



### enumerate AD for SPNS with getUserSPNs.ps1

``` powershell
.\getUserSPNS.ps1
```
